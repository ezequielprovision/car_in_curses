# car_in_curses
Learning class and objects with curses module

![](/car_in_curses.jpg)


## Installation 
```
pip install curses
```

## Instructions
* **s** : initialize car
* **q** : quit 
* **p** : show driver's info
* **up_arrow** : move forward
* **down_arrow** : move backward
* **right_arrow** : turn right
* **left_arrow** : turn left
